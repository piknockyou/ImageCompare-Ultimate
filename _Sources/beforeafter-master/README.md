# BeforeAfter

A browser-based tool for comparing and combining before/after images. No server needed, runs entirely in your browser.

![BeforeAfter Screenshot](screenshot.png)

## Features

- 📸 Combine before/after images side by side or vertically
- 🔍 Visual diff tool to highlight changes between images
- 📋 Paste images directly from clipboard
- 📥 Drag and drop support
- 🎨 Customizable:
  - Background color
  - Label size and color
  - Gap between images
  - Export format and quality
- 💾 Download combined or diff images
- 📱 Mobile-friendly
- 🚀 No installation required

## Usage

1. Visit [BeforeAfter](https://yani-.github.io/beforeafter/)
2. Add images using:
   - Paste from clipboard (Ctrl+V)
   - Drag and drop
   - File browser
3. Customize appearance using the controls
4. Click "Combine Images" or "Show Diff"
5. Download the result

## Keyboard Shortcuts

- `Ctrl+V` - Paste image
- `Ctrl+Enter` - Combine images
- `Delete` - Clear selected area
